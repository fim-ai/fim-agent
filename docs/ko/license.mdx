## 빠른 요약

### 허용됨

| 사용 사례 | 세부 사항 |
|---|---|
| **내부 사용** | 조직 및 계열사 내에서 내부 비즈니스 운영을 위해 FIM One을 사용합니다. |
| **수정** | 소스 코드를 수정하고 파생 저작물을 작성합니다. |
| **배포** | 소프트웨어를 배포합니다(라이선스 사본 포함). |
| **앱에 임베딩** | FIM One을 자체 애플리케이션의 구성 요소로 사용합니다. 단, 경쟁 제품이나 호스팅 서비스가 아닌 경우에만 가능합니다. |
| **단일 테넌트 배포** | 전용 환경에서 단일 고객을 위해 FIM One을 배포합니다. 단, 브랜딩이 유지되고 제3자 액세스가 허용되지 않는 경우에만 가능합니다. |

### 제한사항

| 제한사항 | 세부사항 |
|---|---|
| **멀티테넌트 SaaS / 호스팅 서비스** | FIM One을 제3자에게 서비스로 제공할 수 없습니다(예: "FIM One-as-a-Service", 관리형 멀티테넌트 플랫폼, 또는 클라우드 서비스로 FIM One 기능에 대한 액세스 재판매). |
| **경쟁 제품** | FIM One을 사용하여 경쟁하는 에이전트 오케스트레이션, 워크플로우 실행 또는 작업 계획 제품을 구축하거나 운영할 수 없습니다. |
| **화이트라벨링 / 브랜딩 제거** | FIM One 브랜딩, 로고, 저작권 공지, 또는 라이선스 공지를 제거, 변경 또는 모호하게 할 수 없습니다. 화이트라벨링은 별도의 상용 라이선스가 필요합니다. |
| **라이선스 확인 우회** | 라이선스 검증, 기능 제한 또는 권한 확인을 제거, 비활성화 또는 우회할 수 없습니다. |

## 주요 용어

- **호스팅 서비스** — 네트워크를 통해 제3자가 소프트웨어의 기능에 접근할 수 있도록 하는 모든 서비스(SaaS, PaaS, API 서비스, 관리형 배포).
- **경쟁 제품** — FIM One과 실질적으로 유사한 에이전트 오케스트레이션, 에이전트 런타임, 워크플로우 실행 또는 작업 계획 기능을 제공하는 모든 제품.
- **파생 저작물** — 소프트웨어를 기반으로 하거나 소프트웨어에서 파생된 모든 저작물(수정, 확장, 개조 포함).
