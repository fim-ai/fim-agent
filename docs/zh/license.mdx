## 快速总结

### 允许的用途

| 用例 | 详情 |
|---|---|
| **内部使用** | 在您的组织和关联公司内部使用 FIM One 进行内部业务运营。 |
| **修改** | 修改源代码并创建衍生作品。 |
| **分发** | 分发该软件（包括许可证副本）。 |
| **嵌入到您的应用中** | 将 FIM One 用作您自己应用中的组件，前提是这些应用不是竞争产品或托管服务。 |
| **单租户部署** | 在专用环境中为单个客户部署 FIM One，前提是品牌保持完整且不授予第三方访问权限。 |

### 受限制的使用

| 限制 | 详情 |
|---|---|
| **多租户 SaaS / 托管服务** | 您**不得**向第三方提供 FIM One 作为服务（例如"FIM One 即服务"、托管多租户平台或将 FIM One 功能作为云服务转售）。 |
| **竞争产品** | 您**不得**使用 FIM One 构建或运营竞争性的智能体编排、工作流执行或任务规划产品。 |
| **白标 / 移除品牌** | 您**不得**移除、更改或隐藏 FIM One 品牌、徽标、版权声明或许可声明。白标需要单独的商业许可证。 |
| **规避许可证检查** | 您**不得**移除、禁用或规避任何许可证验证、功能门控或权利检查。 |

## 关键术语

- **托管服务** — 任何允许第三方通过网络访问软件功能的服务（SaaS、PaaS、API 服务、托管部署）。
- **竞争产品** — 任何提供智能体编排、智能体运行时、工作流执行或任务规划功能且与 FIM One 基本相似的产品。
- **衍生作品** — 任何基于或衍生自软件的作品，包括修改、扩展和改编。
