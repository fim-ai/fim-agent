## Schnelle Zusammenfassung

### Zulässig

| Anwendungsfall | Details |
|---|---|
| **Interne Nutzung** | Verwenden Sie FIM One innerhalb Ihrer Organisation und verbundener Unternehmen für interne Geschäftstätigkeiten. |
| **Änderung** | Ändern Sie den Quellcode und erstellen Sie abgeleitete Werke. |
| **Verteilung** | Verteilen Sie die Software (mit einer Kopie der Lizenz). |
| **Einbettung in Ihren Apps** | Verwenden Sie FIM One als Komponente in Ihren eigenen Anwendungen, solange diese kein konkurrierendes Produkt oder gehosteter Dienst sind. |
| **Single-Tenant-Bereitstellung** | Stellen Sie FIM One für einen einzelnen Kunden in einer dedizierten Umgebung bereit, vorausgesetzt, das Branding bleibt erhalten und es wird kein Zugriff durch Dritte gewährt. |

### Eingeschränkt

| Einschränkung | Details |
|---|---|
| **Multi-Tenant-SaaS / Gehosteter Service** | Sie dürfen FIM One nicht als Service für Dritte anbieten (z. B. „FIM One-as-a-Service", verwaltete Multi-Tenant-Plattformen oder Weiterverkauf von Zugriff auf FIM One-Funktionalität als Cloud-Service). |
| **Konkurrierendes Produkt** | Sie dürfen FIM One nicht verwenden, um ein konkurrierendes Agent-Orchestrierungs-, Workflow-Ausführungs- oder Task-Planning-Produkt zu erstellen oder zu betreiben. |
| **White-Labeling / Entfernen von Branding** | Sie dürfen FIM One-Branding, Logos, Urheberrechtshinweise oder Lizenzhinweise nicht entfernen, ändern oder verschleiern. White-Labeling erfordert eine separate kommerzielle Lizenz. |
| **Umgehen von Lizenzprüfungen** | Sie dürfen keine Lizenzverifizierung, Feature-Gating oder Berechtigungsprüfungen entfernen, deaktivieren oder umgehen. |

## Schlüsselbegriffe

- **Gehosteter Dienst** — Jeder Dienst, der es Dritten ermöglicht, auf die Funktionalität der Software über ein Netzwerk zuzugreifen (SaaS, PaaS, API-Dienste, verwaltete Bereitstellungen).
- **Konkurrierendes Produkt** — Jedes Produkt, das Agenten-Orchestrierung, Agenten-Laufzeit, Workflow-Ausführung oder Aufgabenplanungsfunktionen bietet, die FIM One wesentlich ähnlich sind.
- **Abgeleitete Werke** — Jedes Werk, das auf der Software basiert oder von ihr abgeleitet ist, einschließlich Änderungen, Erweiterungen und Anpassungen.
